# Introduction to Mobile Systems - Vehicle Rental System

**Course name:** Introduction to Mobile Systems  
**Set Number:** 4  
**Student name:** Onur Keskin  
**Student ID:** 54825  

### Description
This project is an object-oriented Java console application that simulates a vehicle rental system. It demonstrates the use of abstract classes, interfaces, polymorphism, collection sorting, and XML persistence. The system manages different types of vehicles with specific behaviors.
